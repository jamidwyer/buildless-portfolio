# light portfolio

My portfolio site, trying (and loving!) lit-element web components and minimal bells/whistles.

## Develop

`npm i --save-dev @web/dev-server`

`npx web-dev-server --open --node-resolve`

http://localhost:8000/

## Deploy

Open a PR.
Check the Netlify preview.
Get an approval.
Merge.
